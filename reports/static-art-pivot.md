# Static Production Art Pivot

The 50-character procedural Blender regression is rejected and is not collection art.

Production now uses the polished twelve-character website roster as the mandatory style and quality reference. Every final NFT must be a detailed, full-body, static character render with complete discipline equipment and must match its locked token assignment. Riggability is not a launch requirement.

The collection stays gated until all 1,000 final images exist and pass:

1. visual quality review against the approved roster;
2. image-to-metadata trait verification;
3. sport-equipment completeness checks;
4. exact and perceptual duplicate checks;
5. master and marketplace dimension validation.

`art/static-collection/prompt-manifest.json` is the authoritative one-to-one mapping from token metadata to production art prompts. Blender scripts remain only as non-production experiments.
